# Day 4 — Probability Basics

## Topics Covered
- Basic Probability
- Events and Outcomes

## Key Learnings
- Probability measures likelihood of an event.
- Range: 0 to 1

## Formula
P(A) = Favorable outcomes / Total outcomes

## Example
Probability of getting a 3 on a die:
P(3) = 1/6

## Concepts
- Certain event → Probability = 1
- Impossible event → Probability = 0

## Importance in ML
- Used in predictions
- Forms base of many algorithms

## Insight
Probability is the language of uncertainty.

## Reflection
Every ML model is essentially making probabilistic decisions.
